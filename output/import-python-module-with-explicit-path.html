Import python module with explicit path

Published: Fri 14 March 2014
By dai

In Tips.

tags: module import pip python

Sometimes, when importing a python module into a python script, you get error like this one
File "", line 1, in ImportError: No module name package_name

Check if that package is installed. If it is installed try using something like:
import sys sys.path.append(r"/path_to_package") import package_name
